# **WEB stranica o AI alatima**
Ovo je web stranica izrađena kao projekt u sklopu kolegija na fakultetu.

## **Opis**
Razvijena je responzivna web stranica koja omogućuje pregled i usporedbu popularnih AI alata kao što su ChatGPT, Gemini i Claude. Projekt se sastoji od više povezanih web stranica koje sadrže detaljne informacije o svakom alatu, tablični prikaz njihovih funkcionalnosti te obrazac za unos i prikupljanje korisničkih podataka. Poseban naglasak stavljen je na jednostavnu navigaciju, pregledan dizajn i korisničko iskustvo. 
Stranica je izrađena korištenjem tehnologija HTML, CSS, JavaScript i Bootstrap za izradu responzivnog sučelja. <br>

## **Tehnologije**
- HTML
- CSS
- JavaScript  <br>
